# chrome-plugin-danmu
只有直播没有弹幕，能行吗？今天公司举行了一场大会，我们只能在屏幕前看直播，所以我做了这个插件来给直播加上弹幕.

# 原理
找到视频,然后在视频上面添加一层弹幕遮罩,下面添加一个 input 框,后端统一请求一个 websocket 服务.

# 演示地址
